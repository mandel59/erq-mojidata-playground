# Erq Mojidata Playground

## これは何？

Erq Mojidata Playground は、Erq と Mojidata のデータを使って遊ぶためのプロジェクトです。

このリポジトリをクローンして、ローカルで実行することで、Erq と Mojidata のデータを使って遊ぶことができます。
